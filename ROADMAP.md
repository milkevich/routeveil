# Roadmap

Routeveil development prioritizes lifecycle correctness, accessibility, browser-history safety, package quality, and stable React Router integration.

## Current direction

- Strengthen interruption, cleanup, focus, timeout, and unmount guarantees.
- Expand deterministic lifecycle and consumer compatibility coverage.
- Keep public documentation and machine-readable references aligned with verified behavior.

## Future exploration

- Shared-element transitions built on the hardened lifecycle state machine.
- Additional compatibility work when real consumer fixtures demonstrate support.
- Performance improvements supported by profiling and lifecycle evidence.

Shared-element transitions are not implemented, scheduled, or part of the current public API.
